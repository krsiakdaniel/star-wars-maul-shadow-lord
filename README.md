# Maul: Shadow Lord — Fan Site

Fan site for the Star Wars animated show **Maul: Shadow Lord** (Disney+, 2026).

## About This Project

This is a **fullstack application**. The frontend is a polished Next.js fan site — it demonstrates standard frontend quality but is not the main focus.

The **primary goal** of this repo is the **backend and fullstack integration**:

- REST API built with Next.js API Routes
- API documentation with **Swagger / OpenAPI**
- Demonstrating fullstack patterns: data fetching, typed endpoints, request/response contracts

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)

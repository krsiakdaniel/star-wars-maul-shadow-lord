# Maul: Shadow Lord — Fan Site

Fan site for the Star Wars animated show **Maul: Shadow Lord** (Disney+, 2026).

Project: <https://star-wars-maul-shadow-lord.vercel.app>

## About This Project

> This project is a full-stack application.

The **frontend** is a Next.js site — it meets the usual standards and is of good quality, but is not the main focus.

The **primary goal** of this repo is the **backend** and **full-stack integration**:

- REST API
- API documentation with **Swagger**
- Demonstrating full-stack patterns: data fetching, typed endpoints, request/response contracts

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)
